## 🎯 项目概述

**毛毛AI投研助手 v2.0** 是基于Claude深度评估的**实质重建版本**。与v1.0（文档驱动、模拟数据、虚假成就）不同，v2.0是**代码驱动、真实数据、实质功能**的专业投研系统。

### 🔄 根本转变
- **从**: 50+文件混乱架构，硬编码模拟数据，无测试无配置
- **到**: 清晰四层架构，真实akshare数据，19个单元测试，专业配置中心
- **从**: 3行if-else危险投资建议
- **到**: 多维加权评分系统，强制风险提示

## 🚀 快速开始

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 基本使用
```bash
# 分析单只股票（茅台）
python main.py 600519

# 批量分析并排名
python main.py 600519 300750 000858 002415

# 查看历史评分趋势
python main.py 600519 --trend

# 查看记忆系统概览
python main.py --summary
```

### 3. 运行测试
```bash
# 运行所有单元测试
pytest tests/ -v

# 测试覆盖率报告
pytest --cov=core tests/
```

## 📊 核心特性

### 1. 🎯 真实数据接入
- **多数据源**: akshare（实时数据） + baostock（历史数据）
- **网络容错**: 自动重试 + 缓存机制 + 优雅降级
- **状态透明**: 明确标记数据来源（live/error/fallback）

### 2. 🧮 多维评分系统
```
| 维度 | 权重 | 关键指标 | 评分逻辑 |
|------|------|----------|----------|
| 估值 | 30分 | PE(TTM)/PB | PE≤15得15分，PE>40得2分 |
| 技术面 | 30分 | 趋势/RSI/MACD/量价 | 趋势+RSI+MACD+量价配合 |
| 基本面 | 40分 | ROE/净利增速/毛利率 | ROE≥20得15分，ROE<6得1分 |

| 总分 | 评级 | 建议 |
|------|------|------|
| ≥80 | A+ | 强烈关注 |
| ≥65 | A | 可以关注 |
| ≥45 | B | 持续观察 |
| <45 | C | 暂时回避 |
```

### 3. 📄 专业报告生成
- **终端报告**: 彩色进度条，结构化输出
- **Markdown报告**: 完整分析报告，适合分享
- **JSON报告**: 原始数据，适合程序处理
- **强制风险提示**: 所有报告自动附加免责声明

### 4. 💾 持久化记忆系统
- **有界管理**: 每只股票最多30条记录，总数上限500条
- **自动压缩**: 超过上限自动删除最旧记录
- **趋势分析**: 历史评分趋势，改善/恶化/稳定判断
- **跨次记忆**: 分析记录持久化保存

### 5. 🧪 完整测试套件
- **19个单元测试**: TechIndicators(8) + ScoreEngine(9) + 集成测试(2)
- **真实逻辑测试**: 不使用模拟数据，测试真实业务逻辑
- **测试覆盖率**: 核心模块100%测试覆盖

## 📈 技术指标

### 技术分析指标
- **移动平均线**: MA5, MA20, MA60
- **相对强弱指数**: RSI(14)
- **MACD**: 12/26/9 金叉死叉判断
- **布林带**: 上轨/中轨/下轨，支撑阻力位
- **成交量分析**: 量比，成交量均线
- **趋势判断**: 强势上涨/上涨/震荡/下跌/强势下跌

### 基本面指标
- **盈利能力**: ROE，净利润增速
- **盈利质量**: 毛利率，净利率
- **财务健康**: 资产负债率
- **估值水平**: PE(TTM)，PB，市值

### 数据来源
